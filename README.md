# adhan-esp32cam
adhan times and reminders via esp32-cam, 0.9" i2c older, and passive piezo buzzer via rtttl files.

what is does:

initiates captive portal for wifi connection, 

fetches location based of ftp server, 

disables captive portal / wifi, 

calculated adhan times based on location, 

displays adhan times on screen + 

buzzes 10 minutes before adhan / 

buzzes at adhan time
